# Todo App in Flutter
Simple todo app with recycle bin written in Flutter.

## Demo
![Todo app demo](assets/demo.gif)

## Features
This todo app currently implements the following functionalities:
- Add new todos
- Mark todos as completed (mark again to undo)
- Delete todos
- View deleted todos
